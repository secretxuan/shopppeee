## 📊 数据流

### 用户认证流程
```
登录/注册 → API请求 → 获取Token → 存储Token → 
更新状态 → 跳转页面 → 后续请求携带Token
```

### 商品浏览流程
```
访问页面 → 加载商品 → 渲染列表 → 
用户交互 → 更新状态 → 重新渲染
```

### 购物车流程
```
添加商品 → 更新Store → 保存LocalStorage → 
更新徽章 → 显示提示 → 页面同步
```
